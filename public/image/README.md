# 画像ファイル

このディレクトリには以下の画像ファイルを配置してください：

## 必要なファイル

### 新商品関連
- `new_product_dummy.png` - 新商品のダミー画像（240x192pxまたは適切なサイズ）

### 書籍関連
- `book1.jpg` - 建築設計の基礎
- `book2.jpg` - 構造設計入門  
- `book3.jpg` - 設備設計の基礎
- `book4.jpg` - 建築法規入門
- `book5.jpg` - 建築施工管理

### ソフトウェア関連
- `software1.jpg` - Vectorworks
- `software2.jpg` - Revit

### その他
- `ファビコンb.png` - ファビコン用画像

## 注意事項
- ファイル名は大文字小文字を区別します
- 画像が見つからない場合はプレースホルダーが表示されます
- 書籍・ソフト・AI画像は商品カードで使用されます
- 推奨サイズ：240x192px または適切なアスペクト比

## ファビコンについて
ファビコンファイルは `public/` ディレクトリ直下に配置してください。
